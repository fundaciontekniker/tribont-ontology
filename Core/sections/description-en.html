Core module: Description back to ToC

Details

The Core module includes the TechnicalProperty (i.e., representing qualifiable or quantifiable technical attribute, or characteristics), SystemProperty (i.e., representing qualifiable or quantifiable attribute, or characteristic of a sample system) and OperationProperty (i.e., representing quantifiable operation attribute, or characteristic) classes which are specialisations of the Property class. Each of these classes in turn includes several specialisations that are further re-specialised. For example: the OperationalProperty class includes as specialisation, along with others, the EnvironmentProperty class which in turn includes the Pressure, Humidity, Temperature, etc. classes.

Furthermore, the Core module includes the Body class (i.e., representing an aggregate of particles to which it is possible to apply a force) that is a specialisation of the SubjectOfInterest class and can be characterised (isCharacterisedBy object property) by a set of properties (i.e.BodyProperty class).

In addition, it includes Role, Company (and its Laboratory specialisation), Document and Standard classes, as well as some relevant object properties that are used in several modules.
